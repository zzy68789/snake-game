# 🐍 贪吃蛇游戏

一个精美的HTML5贪吃蛇游戏，支持键盘控制和最高分记录。

## 🚀 快速开始

### 方法1：独立版本（推荐）
双击 `start_standalone.bat` 启动游戏，无需服务器！

### 方法2：服务器版本
双击 `start_game.bat` 启动本地服务器，然后访问游戏。

## 📁 项目结构

```
white/
├── src/                    # 源代码目录
│   ├── js/game.js          # 游戏逻辑
│   ├── css/style.css       # 游戏样式
│   └── html/
│       ├── index.html      # 主页面（需要服务器）
│       └── snake_game_standalone.html  # 独立版本
├── docs/README.md          # 详细文档
├── assets/                 # 资源文件目录
├── start_game.bat          # 服务器启动器
├── start_standalone.bat    # 独立版本启动器
└── README.md              # 本文件
```

## 🎮 游戏特色

- ✨ 流畅的游戏体验和精美界面
- 🎯 支持方向键和WASD控制
- 🏆 自动保存最高分记录
- 📱 响应式设计，支持移动设备
- ⏸️ 暂停/继续功能
